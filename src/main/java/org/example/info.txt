Основні анотацій:

@Entity:         позначає клас як сутність (entity), яка буде зберігатися у базі даних.
@Table:          вказує назву таблиці, до якої мапується клас.
@Id:             позначає первинний ключ у таблиці.
@GeneratedValue: вказує стратегію генерації значень для первинного ключа.
@Column:         вказує назву стовпця в таблиці для конкретного поля класу.

Hibernate потребує драйвера для підключення до бази даних.
Драйвер забезпечує зв'язок між вашим додатком і базою даних.
    Для PostgreSQL це org.postgresql.Driver.
    Для MySQL — com.mysql.cj.jdbc.Driver.
